# Github Search

### Descrição

 é um projeto pensado para pesquisar repositórios públicos de um organização no Github. Seu objetivo é encontrar a encontrar os repositórios da organização que for digitada pelo usuário, serão exibidos até 50 repositórios na tela. Caso a página não seja encontrada, será exibida uma mensagem na tela.

O projeto utiliza: React e React-Router-DOM.
